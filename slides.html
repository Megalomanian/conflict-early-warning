Motivation

Problem: Social media discussions can escalate from civil debate to hostile conflict within hours — but existing tools only analyze after escalation occurs.

Current Limitations

  • Gap 1 Post-hoc sentiment analysis — too late for intervention
  • Gap 2 Instance-centric risk scoring — no temporal modeling
  • Gap 3 No early warning — detection without forecasting

Our Solution

  • C1 Weakly supervised conflict index
  • C2 CNN-BiLSTM temporal forecasting
  • C3 Practical early-warning triggers

Conflict Index Formulation

Weakly supervised: No manual labels needed

\( c_{t,i} = \sigma(w_a \tilde{a}_{t,i} + w_e \tilde{e}_{t,i} + w_s \tilde{s}_{t,i} + b) \in [0,1] \)

ComponentSignalImplementationWeight
Attack \(a_{t,i}\)Antagonism, threats, abuseBERT sentiment (1★ → attack)0.5
Emotion \(e_{t,i}\)Negativity + high arousal\(\alpha p_{neg} + (1-\alpha)p_{anger}\)0.3
Stance \(s_{t,i}\)Semantic polarizationEmbedding + k-means clustering0.2

🔑 Key design: ECDF calibration → sigmoid fusion → top-\(k\) bin aggregation (\( \rho = 0.15 \))

CNN-BiLSTM Architecture

Design

  • 1D Conv Front-end
    Kernel sizes 3 & 5, 32 channels
    → Multi-scale local patterns
  • 2-Layer BiLSTM
    64 hidden units/direction, dropout 0.2
    → Long-range dependencies
  • Linear Projection Head
    Concat last-forward + first-backward
    → Direct multi-step forecast (H=6)

Training

  • Loss: Huber (\(\delta = 0.5\))
  • Optimizer: Adam (LR \(10^{-3}\))
  • Early stopping (patience=40)
  • Input: \(L=12\) bins (6 days @ 12h)
  • Output: \(H=6\) bins (3 days ahead)

Why CNN+BiLSTM?
CNN captures local trends & bursts; BiLSTM models persistence & long memory cycles.

Early-Warning Trigger Rules

RuleMechanismPurpose
R1: Threshold\(\max_h \hat{c}_{t+h} \ge \eta\)Absolute risk exceedance
R2: Rapid Growth\(\hat{c}_{t+H} - \bar{c}_t \ge \gamma\)Fast-rising trends
R3: Persistence\(\sum_{j=0}^{K-1} \text{Warn}_{t-j} \ge K\)Filter transient noise
R4: Tail-AwareExtreme-tail intensity \(u_t\)Small fraction driving escalation
R5: CompositeWeighted score \(S_t\)Flexible precision–recall trade-off

Inspired by EMBERS (KDD 2014, Test of Time Award) — operational forecasting with structured lead-time guarantees.
